# ⚔️ GitHub Fight — Developer vs Developer

একদম নতুন ধরনের **Interactive GitHub Comparison Tool**। দুইজন ডেভেলপারের মধ্যে যুদ্ধ লাগিয়ে দাও!

### 🔥 ফিচারসমূহ
- রিয়েল-টাইম পাশাপাশি তুলনা
- স্মার্ট স্কোরিং সিস্টেম
- ট্রফি অ্যানিমেশন + কনফেটি
- Data Saver Mode (মোবাইল নেটওয়ার্কের জন্য)
- Fully Responsive & Beautiful UI

**Tech Stack**: HTML5, CSS3, Vanilla JavaScript, GitHub REST API, Tailwind CSS
